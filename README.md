# 205-Degrees-Smoke-Shack
BBQ Website design for 205 Degrees Smoke Shack

## Local preview

```bash
npm run dev
```

## Screenshots

The screenshot task captures a browser-rendered PNG when Chrome/Chromium is available. If your environment does not already include a browser executable, run:

```bash
npm run setup:browser
```

You can also point the capture script at a custom browser binary:

```bash
BROWSER_EXECUTABLE=/path/to/chrome npm run screenshot
```

When a browser cannot be found, `npm run screenshot` still writes `screenshots/homepage.svg` as a fallback preview.
